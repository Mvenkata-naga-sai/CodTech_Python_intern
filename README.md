
# 🤖 AI Sentiment Analyzer

## 🚀 Overview
An AI-powered Sentiment Analysis tool built using Python and Natural Language Processing (NLP) to classify text into Positive, Negative, or Neutral sentiments.

---

## 📌 Features
- 🧠 Sentiment classification (Positive / Negative / Neutral)
- ✍️ User input text analysis (CLI-based)
- 📊 Confidence score output
- ⚡ Fast and lightweight
- 🔄 Real-time processing

---

## 🧠 Tech Stack
- **Language**: Python  
- **Libraries**: TextBlob, NLTK  
- **Concepts**: Natural Language Processing (NLP)

## ⚙️ Installation & Setup
1. Clone Repository  
git clone https://github.com/your-username/ai-sentiment-analyzer.git  

2. Install Dependencies  
pip install -r requirements.txt  

3. Download NLTK Data  
python -c "import nltk; nltk.download('punkt')"  

4. Run  
python app.py
