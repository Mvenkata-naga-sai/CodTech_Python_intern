# 📊 Real-time Dashboard (Streamlit)

🚀 A dynamic and interactive **Real-time Dashboard** built using **Streamlit**, designed to visualize live data streams with customizable controls and modern UI components.

---

## 📌 Features

* 📈 Live Data Streaming
* 📊 Interactive Charts (Plotly)
* 🔄 Auto Refresh Functionality
* 🎛️ Sidebar Controls (Refresh Rate & Data Points)
* 📋 Real-time Data Table
* ⚡ Lightweight & Fast Execution

---

## 🧠 Tech Stack

* **Framework**: Streamlit
* **Data Processing**: Pandas
* **Visualization**: Plotly
* **Language**: Python

---

## 🏗️ Project Structure

```bash
realtime-dashboard/

├── app.py                    # Main Streamlit app
├── utils/
│   └── data_generator.py     # Generates live data

└── README.md
```

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/realtime-dashboard.git
cd realtime-dashboard
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Application

```bash
streamlit run app.py
```

---

## 🌐 How It Works

* The application continuously generates new data points using a custom Python function.
* Data is appended in real-time and visualized using a line chart.
* Users can control refresh speed and data size from the sidebar.
* The dashboard updates automatically without manual reload.
