
# 📊 Real-time Dashboard (Streamlit)

## 🚀 Overview
A real-time interactive dashboard built using Streamlit for live data visualization.

## ⚙️ Setup
pip install -r requirements.txt
streamlit run app.py

## 📌 Features
- Live data streaming
- Dynamic charts
- Sidebar controls

## 📌 Conclusion
This project demonstrates real-time data visualization using Streamlit.
